# Maithili News Frontend

Multi-page, responsive newsroom frontend with admin/reporter panels and local QA automation.

## Tech
- HTML5
- Modern CSS
- Vanilla JavaScript
- Playwright (E2E/health tests)

## Run locally
```powershell
node local-server.js
```

Open:
- `/home/`
- `/admin-login/`
- `/admin-panel/`
- `/reporter-panel/`
- `/article-page/`

## QA
```powershell
npm run qa:test
```
